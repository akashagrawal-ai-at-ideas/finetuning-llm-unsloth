# Fine-tuning LLM Unsloth

This repository is dedicated to fine-tuning large language models (LLMs), specifically focusing on the fine-tuning of Llama 3. 

## Getting Started

### Prerequisites
- Python 3.8+
- PyTorch 1.10+
- Transformers library

### Installation

Clone the repository:
```bash
git clone https://github.com/akashagrawal-ai-at-ideas/finetuning-llm-unsloth.git
```

Navigate to the repository:
```bash
cd finetuning-llm-unsloth
```

Install the required packages:
```bash
pip install -r requirements.txt
```
